Краткая документация по программе AtoZ [РУС]
Данная программа предназначена для быстрого перевода введённого текста в другой регистр или исправлении ошибки раскладки клавиатуры(русская\английская).

Режимы работы:
- Смена языка(по умолчанию при запуске ПО)
Изменяет фразу введённую на руском на английской раскладке на русскую раскладку с учётом регистра, и наоборот. Также, можно настроить какой символ на какой заменить. Для этого необходимо произвести изменения в файле lang.dat, который идёт вместе с исполняемым файлом, и имеет кодировку Unicode, что позволяет подставить туда практически любой возможный символ.

Строение lang.dat:
Это единственный дополнительный файл программы(не учитывая иконку и документацию), состоящий из данный следующего формата: <Введённый символ> <На какой символ заменить>.
Строки в которых не 1 пробел - будут игнорироваться. 
Чтение производиться только первого символа лексемы, и если будет введено слово - то будет прочтён только первый символ.
Идентичные ключевые символы - игнорируются, даже если ключи разные.
По умолчанию работает и со символами которые соответствуют комбинации клавиш Shift + <цифра>, такие как: ", ;, #, &, и т.п.
В целях оптимизации не допускайте строки где основное и изменяемое значения - идентичны.

Если программа не нашла значение из lang.dat - то она запишет в результат исходный символ.


- Смена регистра букв
Данная функция не требует дополднительных файлов, и работает напрямую с .NET Framework, тем самым поддерживает регистр различных языков. Регистр изменяеться на противоположный, что очень удобно если Вы по случайности не заментили нажатый на своей клавиатуре Caps Lock.

Комбинации клавиш:
- Для указания того, что Вы хотите переключиться на смену языка - Вы можете нажать как соответствующий пункт меню, так и комбинацию Ctrl + Q.
- Для указания того, что Вы хотите переключиться на смену регистра - Вы можете нажать как соответствующий пункт меню, так и комбинацию Ctrl + E.
- Очистка буфера ввода возможна как путём удаления текста из него(выделить и удалить, зажать кнопку удаления на клавиатуре), так и нажатием соответствующей кнопки в меню, либо комбинацией Ctrl+ Del.
- Закрытие приложения - нажатием соответствующей кнопки окна, меню, либо комбинацией Alt + F4.

Для удобности использования - выделение всего текста в окне ввода можно осуществить двойным кликом. Двойной клик построке с результатом конвертации - не только выделит её, но и скопирует всё содержимое в буфер обмена.


Ошибки:
Если программа не запускается, либо при запуске сразу возникает ошибка - то скорее всего вам требуеться установить платформу .NET Framework, сделать это можно по ссылке на официальный сайт Microsoft: https://www.microsoft.com/ru-RU/download/details.aspx?id=55167. Для большей совместимости и быстродействия программа использует .Net Framework 2.0, соответственно, она совместима со всеми более новыми версиями.

Если отсутствует файл lang.dat - то выведеться соответствуюшее предупреждение, и будет возможна только работа с изменением регистра букв.


Минимальные системные требования:
- .NET Framework 2.0
- Минимум 6.8 Мб ОЗУ для запуска. С полностью саполненным буфером в 65535 символв - 10.4 МБ
- Windows XP x86
- 1 ядерный процессор 1 ГГц
